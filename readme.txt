Displays a random image or a slider of images.

== Description ==

You can upload images right from the widget settings page, using the built-in Media Uploader. You can also resize the image to your desired dimentions.
The script for the slider is only queued if the widget is active and slider on that widget is enabled.


== Installation ==

1. Upload the whole plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Appearance -> Widgets and add an 'Image Rotator' widget to a widget area.
5. Enjoy!
